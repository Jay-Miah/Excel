# 📊 Excel Retail & Sales Analysis Project  
*A Data Technician Bootcamp Project*

This project showcases my ability to analyse, clean, and visualise retail and sales data using **Microsoft Excel**. It demonstrates core data skills developed during my Data Technician bootcamp, including formula‑based analysis, PivotTables, and dashboard creation.

---

## 🚀 Project Overview
The objective of this project was to transform raw retail and sales data into meaningful insights. I used Excel to clean the dataset, apply analytical formulas, and build interactive PivotTables and charts that highlight key performance trends across products, stores, and time periods.

---

## 🧠 Key Skills Demonstrated

### 🔢 Excel Formulas & Functions
Applied a wide range of functions to automate calculations and extract insights:

- **SUM**, **SUMIF** — total and conditional sales calculations  
- **AVERAGE**, **AVERAGEIF** — performance and trend analysis  
- **DATE**, **MONTH**, **YEAR** — time‑based data extraction  
- **UNIQUE** — identifying distinct products and customer groups  
- **VLOOKUP** — retrieving product details and merging datasets  
- **SWITCH** — logic‑based categorisation for cleaner reporting  

---

### 🧹 Data Cleaning & Preparation
- Filtering and sorting large datasets  
- Removing duplicates and inconsistencies  
- Standardising date formats  
- Creating calculated fields for deeper analysis  

---

### 📊 PivotTables & Reporting
Built interactive PivotTables to summarise:

- Sales by product category  
- Monthly revenue trends  
- Store‑level performance  
- Customer purchasing behaviour  

Developed PivotCharts to visualise patterns and support decision‑making.

---

### 📈 Data Visualisation
Designed clear, professional charts including:

- Line charts for monthly sales trends  
- Bar and column charts for product performance  
- Pie charts for category distribution  
- Slicers for interactive filtering and exploration  

---

## 🎯 Key Outcomes
- Delivered a complete Excel workflow from raw data to dashboard  
- Demonstrated strong competency in formulas, PivotTables, and visualisation  
- Gained hands‑on experience with retail and sales datasets  
- Built a portfolio‑ready project showcasing practical analytical skills
